### 步驟 0：確認前提（可選）

```

→ 預期結果說明
→ 若結果異常：處理方式
```

---

### 步驟 2：步驟名稱

<!-- ▶ 需使用者確認時，明確標出等待點：-->

> 以上計畫是否確認？確認後繼續執行。

### 步驟 N：產出報告

完成後以下列格式輸出：

```
✅ 操作完成！

📊 統計：
  項目 A：N 個
  項目 B：N 個

📝 明細：
  - 具體項目 1

⚠️ 需人工確認：
  - （若有）問題描述
```
