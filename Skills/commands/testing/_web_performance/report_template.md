## MD 報告檔格式（步驟 4）

```markdown
# 前端效能檢測報告

- **網址**：{URL}
- **頁面標題**：{title}
- **檢測日期**：{date}
- **檢測工具**：Playwright MCP + Network Analysis
- **外部評分工具**：{PageSpeed API / Lighthouse CLI / CrUX API / 均不可用}

## 1. 載入時間指標
（含評級標準對照表）

## 2. 資源分佈
（含各類型數量、大小、佔比）

## 3. DOM 結構分析
（圖片問題、render-blocking 資源清單）

## 4. 隱私權合規檢查
（<head> 載入順序、網路請求時序、修正建議含程式碼）

## 5. 發現的問題（依嚴重度排序）
（每個問題含：問題描述、影響指標、具體修正方式）

## 6. 優化行動清單
（含優先順序、預估效益、難度）

## 7. 具體修正範例
（每個修正含修正前/修正後的程式碼）

---
*報告由 Claude Code /web_performance Skill 產出*
```
