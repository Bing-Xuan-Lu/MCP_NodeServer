這個 MCP 伺服器能幫你做什麼？

這是您的專屬開發助理。我可以直接讀寫您的專案目錄、連線資料庫執行 SQL、操作 Docker、打包部署，甚至是自動化整理文件與書籤。
您可以直接在對話中提出需求，或者參考下方的 快速指令 (Snippets) 來啟動特定的工作流程。

🏠 助理員工的辦公環境準備 (Setup SOP)

  • Git & Node.js 18+ 安裝並加入系統 PATH
  • Docker Desktop 必須安裝並啟動
  • 書籤工具預設路徑：C:\Users\YourName\AppData\Local\Google\Chrome\User Data\Default
  • Windows 執行 setup.bat 若被攔截請用管理員權限
  • 工作目錄預設 D:\Project，若無 D 槽請修改 config.js
  • 建議安裝 MCP Inspector 擴充功能除錯
